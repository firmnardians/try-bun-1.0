# bun - Hidup berjalan seperti...

To install dependencies:

```bash
bun install
```

To build:

```bash
bun build-file
```

To run server:

```bash
bun run-server
```

To read a file:

```bash
bun read-text
```

To console a react component

```bash
bun react-console
```

To crate a file

```bash
bun create-log
```

To hash password

```bash
bun hash-password
```

This project was created using `bun` in bun v1.0.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
